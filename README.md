# JumpToSpring

## Git commit convention
| 타입 | 내용 |
| --- | --- |
| ✨feat | 새로운 기능 추가 |
| ♻refactor | 코드 리펙토링 |
| 🐛fix | 버그/오류 수정 |
| 📝docs | 문서 수정 |
| 💄style | 코드 포맷 변경, 세미콜론 누락, 코드 변경 없음 |
| design | 사용자 UI 디자인 변경 (CSS 등) |
| move | 코드나 파일의 이동 |
| rename | 이름 변경 |
| remove | 삭제 |
| 🚀chore | 빌드 업무 수정, 패키지 매니저 수정, Production Code와 무관한 설정들 |
| 💡comment | 주석 추가 및 변경 |
